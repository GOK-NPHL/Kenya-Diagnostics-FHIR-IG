This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Behavior: Capability Statements

The following artifacts define the specific capabilities that different types of systems are expected to have in order to comply with this implementation guide. Systems conforming to this implementation guide are expected to declare conformance to one or more of the following capability statements.

Kenya Diagnostics IG Capability Statement

Defines the expected capabilities of this IG.

Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide.

KenyaDiagnosticReport

This profile defines a diagnostic report to be used in a laboratory test.
KenyaDiagnosticServiceRequest

This profile defines a service request to be used in a laboratory test.
KenyaEncounter

This profile defines an encounter that resulted in a laboratory test.
KenyaHealthProviderOrganization

Profile for an Organization with an MFL code as the primary identifier.
KenyaObservation

This profile defines an observation to be used in a laboratory test.
KenyaSpecimen

This profile defines a specimen for a laboratory test.
Patient (IPS)

This profile represents the constraints applied to the Patient resource by the International Patient Summary (IPS) FHIR Implementation Guide and describes the minimum expectations for the Patient resource when used in the IPS composition or in one of the referred resources.

Terminology: Value Sets

These define sets of codes used by systems conforming to this implementation guide.

ValueSet: Justification options for Lab Orders

Justification options for lab orders in this IG
ValueSet: Rejection reasons for Lab Orders

Rejection reasons for lab orders in this IG
Yes/No Value Set

The SNOMED CT codes to answer in the affirmative or negative.

Terminology: Code Systems

These define new code systems used by systems conforming to this implementation guide.

CodeSystem: Laboratory Order Justifications

The reason for ordering a laboratory test.
CodeSystem: Laboratory Order Rejection Reasons

The reason for rejecting a laboratory Order.
KMHFL Code System

The KMHFL code system is a code system that defines the KMHFL identifier.
